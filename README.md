# StockMarketU - Frontend

## Descripción
Frontend para la aplicación StockMarketU, una plataforma de simulación de mercado de acciones que permite ver, comprar y gestionar acciones en tiempo real.

## Tecnologías
- React
- Vite
- Auth0 (autenticación)
- CSS personalizado
